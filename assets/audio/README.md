# Goblin OST background music

The site plays **Stay With Me** from the *Goblin* (도깨비) soundtrack when the music button is tapped.

## Setup (one time)

1. Get the song legally (Apple Music, Spotify download, CD, etc.).
2. Save the MP3 in this folder as:

   **`goblin-stay-with-me.mp3`**

   (Instrumental version works great too.)

3. Refresh the website and tap **Goblin OST** in the top-right corner.

## Alternate filenames

If your file has a different name, either rename it or change `musicUrl` in `js/config.js`.

## Deploying

When you upload to Netlify/GitHub Pages, include the `assets/audio/` folder with the MP3 file so music works on the live link.
